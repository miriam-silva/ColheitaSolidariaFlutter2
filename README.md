# 🥕 Colheita Solidária – Mobile App

O **Colheita Solidária** é um aplicativo mobile desenvolvido para facilitar a doação de alimentos excedentes por agricultores a pessoas em situação de vulnerabilidade social.

O aplicativo foi desenvolvido em **Flutter**, utilizando **Firebase** como backend principal e **Supabase** para armazenamento de imagens das doações.

---

# 📚 Sobre o Projeto

Projeto acadêmico desenvolvido no **4º semestre do curso de DSM – Fatec Matão**, com o propósito de promover solidariedade e sustentabilidade através da redistribuição de alimentos.

---

# 🎯 Objetivos (modelo SMART)

## **S – Specific (Específico)**
Conectar agricultores e recebedores para facilitar a doação de alimentos.

## **M – Measurable (Mensurável)**
Meta alcançada através das funcionalidades de login, cadastro, registro e visualização de doações.

## **A – Achievable (Atingível)**
A equipe possui domínio das tecnologias utilizadas no desenvolvimento do aplicativo.

## **R – Relevant (Relevante)**
Reduzir o desperdício de alimentos e auxiliar pessoas em situação de vulnerabilidade social.

## **T – Time-bound (Temporal)**
Conclusão prevista para o 1º semestre de 2025.

---

# 🚀 Funcionalidades Principais

✅ Login e autenticação de usuários  
✅ Cadastro de Agricultores, Recebedores e Administradores  
✅ Registro de doações com imagem, validade e quantidade  
✅ Visualização de doações disponíveis  
✅ Solicitação de doações  
✅ Gerenciamento administrativo do sistema  
✅ Upload de imagens utilizando Supabase  
✅ Integração completa com Firebase  
✅ Interface intuitiva e responsiva para dispositivos móveis  

---

# 🔥 Backend e Armazenamento

O aplicativo utiliza o **Firebase** como backend principal, sendo responsável por:

- Autenticação de usuários
- Armazenamento de dados
- Controle das doações
- Gerenciamento das solicitações
- Persistência das informações em tempo real

Além disso, o projeto utiliza o **Supabase Storage** para:

- Upload das imagens das doações
- Armazenamento das fotos cadastradas no aplicativo
